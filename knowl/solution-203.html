Solution 5.1.4.5.1.

Supposons que \(a\equiv 0 \mod 2.\) Par les proporiétés de l'équivalence modulo \(2,\) on a que

\begin{align*} a^m+a^n \amp\equiv 0^m + 0^n \mod 2\\ \amp\equiv 0 + 0 \mod 2\\ \amp\equiv 0 \mod 2 \end{align*}

d'où \(a^m+a^n\) est pair. On remarque ici qu'on a utilisé le fait que \(m\) et \(n\) sont non null, car \(0^0\) n'est pas défini.

Supposons maintenant que \(a\equiv 1 \mod 2.\) Par les proporiétés de l'équivalence modulo \(2,\) on a que

\begin{align*} a^m+a^n \amp\equiv 1^m + 1^n \mod 2\\ \amp\equiv 1 + 1 \mod 2\\ \amp\equiv 0 \mod 2 \end{align*}

d'où \(a^m+a^n\) est pair.

Puisque \(a\) est nécessairement pair ou impair, on a montrer dans tous les cas que \(a^m+a^n\) est pair.
